# Sean Coleman – Analytics Portfolio

## 🏦 Associated Bank Loan Default Prediction
**Tools:** Python, XGBoost, Scikit-learn, GridSearchCV  
**Goal:** Predict whether bank customers will repay their loans  

**Key Results:**
- 🏆 XGBoost achieved **0.92 ROC AUC** (up from 0.73 baseline)
- Engineered features: debt burden, income-to-loan ratio, interest-to-income ratio
- Tuned with 5-fold cross-validation to prevent overfitting
